1. Objetivo del Proyecto

Sitio web corporativo de B2 Domus, empresa colombiana especializada en diseño, fabricación e instalación de mobiliario a medida en Bogotá, Funza y alrededores.

🎯 Propósito principal: Escaparate comercial para captar clientes interesados en mobiliario personalizado. No se muestran precios; todo lleva a cotización vía WhatsApp o formulario de contacto.

🤖 Asistente virtual: "Misi" — gatica IA con respuestas automáticas (OpenAI o fallback predefinido).

🎨 Inspiración de diseño: oblikuall.com — estética moderna, minimalista, arquitectónica, premium.

?>

<?php require_once 'includes/footer.php'; ?>  // Footer + scripts

6.2 Navbar con Scroll State

GSAP detecta el scroll y aplica la clase .navbar-scrolled al nav.

gsap.to(nav, {
    scrollTrigger: {
        trigger: 'body',
        start: '80px top',
        toggleClass: { targets: nav, className: 'navbar-scrolled' }
    }
});

Estados CSS:

  • Transparente: background: linear-gradient(to bottom, rgba(29,55,46,0.85), transparent) — Logo y nav-links en blanco
  • Scrolled: background: rgba(245,238,220,0.97) con blur — Logo original, nav-links en verde oscuro

6.4 Chatbot Misi

Botón flotante dorado en la esquina inferior derecha (encima del WhatsApp float).

Flujo:

  1. Usuario escribe mensaje o selecciona sugerencia rápida
  2. JS envía POST a api/misi.php con { message: "..." }
  3. misi.php intenta consultar OpenAI API (si configurada)
  4. Si no hay API key o falla → JS usa respuestas predefinidas por keywords

Respuestas predefinidas (fallback) — 7 categorías:

KeywordsRespuesta
hola, buenas, hey, misiSaludo de bienvenida
servicio, cocina, closet, muebleLista de 5 servicios + invitación a cotizar
cotización, precio, cuántoDerivar a WhatsApp 317 553 1220
proceso, pasos, fabricaciónExplicación del proceso 5 pasos
contacto, teléfono, whatsappDatos de contacto
nosotros, quienes, historiaHistoria de B2 Domus
gracias, ok, perfectoAgradecimiento + oferta de ayuda adicional

Configuración API (opcional):

// En includes/config.php:
define('OPENAI_API_KEY', 'sk-proj-...');  // Tu API key de OpenAI
define('OPENAI_MODEL', 'gpt-4o-mini');    // Modelo

6.5 Formulario de Contacto

  • Método POST a sí mismo (contacto.php)
  • Validación: nombre, email (formato) y mensaje obligatorios
  • Envío vía mail() de PHP a contacto@b2domus.com
  • Éxito: muestra mensaje de confirmación
  • Error: muestra alerta con el problema
